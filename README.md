# peerlink

PeerLink is 'peer to peer middleware'.  Glue to connect peer tools to peer to peer protocols, data and services.

connections:
Hypercore
Network Library
safeFLOW - entity component system for data and simulations

coming soon:
Safe Network
IPFS
Ocean Protocol


Installation:
==============

```
npm install
```

### Run
```
node index.js
```
